# ☁️ CloudCart - Cloud-Based E-Commerce Platform

## Overview

CloudCart is a cloud-based e-commerce web application developed using modern full-stack and cloud technologies. The project demonstrates containerized deployment, cloud hosting, database integration, and scalable architecture.

## Features

* Product listing and browsing
* Responsive user interface
* REST API integration
* MongoDB Atlas database
* Docker containerization
* AWS EC2 deployment
* Amazon S3 image storage
* Frontend and backend integration

## Deployment

Hosted on AWS EC2 using Docker containers.

Frontend Port: 3000
Backend Port: 5000
Database: MongoDB Atlas
Storage: Amazon S3
